# Student Depression Study Website

This website presents an interactive story exploring patterns of student depression, based on datasets including academic pressure, lifestyle factors, and financial stress. It includes interactive Tableau charts, a recommendations section, and a dashboard to visualize insights.

---

## Features

- **Introduction & Problem Statement**: Overview of the student mental health crisis.
- **Interactive Visuals**: Three Tableau charts with insights, filterable and interactive.
- **Recommendations**: Key intervention strategies presented in card-style layout.
- **Interactive Dashboard**: Centered dashboard summarizing key metrics.
